# Arduino PM2.5 LINE Alert

ESP32 + PMS sensor project with LED, relay, buzzer, and LINE push notification.

## Download Sketch

- [Download Arduino_PM25_LINE_Alert.ino](https://raw.githubusercontent.com/ezoigz/Arduino_Pm2.5_-line/main/Arduino_PM25_LINE_Alert.ino)

## Quick Use

1. Open Arduino IDE.
2. Download and open `Arduino_PM25_LINE_Alert.ino`.
3. Install required libraries: `PMS`.
4. Upload to ESP32.
